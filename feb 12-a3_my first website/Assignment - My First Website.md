**Assignment Name:** My First Website
**Description:** Create a personal profile webpage using headings, images, links, and lists.
**Date:** 12th February 2026 (Thursday)

## Objective

To build a personal profile webpage that demonstrates understanding of fundamental HTML elements including headings, images, links, and lists, along with basic CSS styling concepts covered up to 12th February.

## Page Structure

### 1. Navigation Bar
- Name/logo on the left
- Links to each section on the right (About, Skills, Projects, Hobbies, Contact)
- Uses flexbox for layout
- Anchor links (`<a href="#id">`) to jump to sections

### 2. Hero Section
- `<h1>` — Name: **Swara Shetty M**
- Profile placeholder image with `<img>` tag
- Tagline paragraph

### 3. About Me
- `<h2>` section heading with styled bottom border
- White card container with paragraph text introducing myself
- College, CGPA, and career aspiration

### 4. Skills
- `<h2>` section heading
- Unordered list (`<ul>`) with `<li>` items:
  - Languages: Python, C, Java, HTML, CSS
  - Libraries: Pandas, NumPy, Matplotlib
  - Tools: VSCode, Jupyter Notebook, Eclipse, GitHub
  - Soft Skills: Team Player, Leadership, Communication, Punctuality

### 5. Projects
- `<h2>` section heading
- Two project cards, each with:
  - `<h3>` project title
  - `<p>` project description

### 6. Hobbies & Interests
- `<h2>` section heading
- Unordered list: Music, Dance, Reading, Working with AI Tools
- Four placeholder images (`<img>` tags) for each hobby

### 7. Connect with Me
- `<h2>` section heading
- Links using `<a>` tags:
  - Email (mailto link)
  - GitHub profile link

### 8. Footer
- Copyright text

## CSS Styling Applied

- **Font:** Segoe UI sans-serif font family
- **Colors:** Indigo (#4F46E5) accent color, light gray (#f0f0f5) background, white cards
- **Box Model:** Padding, margin, and border used throughout for spacing
- **Flexbox:** Used in navbar layout and hobby images row
- **Border:** Section headings have a 3px solid indigo bottom border
- **Border-radius:** Rounded corners on cards (10px) and profile image (50% for circle)
- **Box-shadow:** Subtle shadow on cards for depth
- **Media Query:** One `@media (max-width: 600px)` for mobile responsiveness
